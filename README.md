## Como usar:

1. Clonar el repositorio
2. Clonar OVMF: [https://github.com/Absurdponcho/OVMFbin]
3. Modificar GNUmakefile para que apunte a tu gdb, gcc, qemu, etc.
3. ejecutar `make setup`
4. ejecutar `make exp`

## Requisitos:

- qemu
- gcc
- make
- gdb
